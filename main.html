Introduction

HTML

3 Parts of HTML

There are 3 main parts to coding websites in HTML:

  • HTML (text)
  • CSS (make it pretty)
  • JS (interacting)

Today we will be focusing on HTML...

HTML Structure

Always start code by typing this structure:
<!DOCTYPE>
<html>

<head>
</head>
<body>
</body>

</html>

Anything that inside < > is called a tag (see next section).

The body tag is where you will code things that users will see. (All the text you see on the screen)

The head tag is where you will code things that users can not see. (colors, interactive components)

Tags

What are tags?

Tags are used to display and format the webpage.

Example of tags

  • <body> - holds elements that are displayed on the page
  • <h1> - displays a heading
  • <img> - displays an image

All tags start with <tagname> and end with </tagname>

HTML Styles

Instead of learning CSS, we will be learning how to make things pretty with HTML styles

To use CSS properties in HTML, you define a style attribute in the element tag that you wish to modify

Example:

<div style="property:value;"></div>

Inside the style attribute, define a property and a value sepreating the two with a colon. Don't forget to put them in quotes!

Formating

Heading Tags

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>

These tags are used to create headings

Example code:

<h1>Look! I'm coding!</h1>

Output:

Look, I'm coding!

There are 6 heading tags: h1 is the largest, h6 is the smallest

Div

<div>

This tag is used hold text and other elements (helps with organization and grouping)

Example code:

<div>
 <h1>Look! I'm coding!</h1>
 <p>Hello World<br>Goodbye World<br>!!!</p>
</div>

Output:

Look! I'm coding!

Hello World
Goodbye World
!!!

You can not see the div that was created, however! Divs are incredibly important for organization!
